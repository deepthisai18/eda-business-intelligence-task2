TASK 2: Exploratory Data Analysis (EDA) & Business Intelligence (BI)

Project Overview
This project performs Exploratory Data Analysis (EDA) and Business Intelligence (BI) on a student dataset using Python. The objective is to analyze the dataset, understand patterns, and generate insights using statistical analysis and data visualization.

Tools and Technologies Used
Python
Pandas
Matplotlib
CSV Dataset
Visual Studio Code

Dataset Description
The dataset contains student information with the following columns:

Name – Name of the student
Age – Age of the student
Marks – Marks obtained by the student

Example Dataset

Name, Age, Marks
Anu, 20, 85
Ravi, 21, 90
Sita, 19, 78
John, 22, 88
Priya 20, 92

Exploratory Data Analysis (EDA)
EDA is performed to understand the dataset structure and statistics.

The following operations were performed:

1. Load dataset using pandas
2. Display first rows using head()
3. Check dataset information using info()
4. Generate statistical summary using describe()

Business Intelligence (BI)
Business Intelligence techniques were applied using data visualization.

The following visualizations were created:

Bar Chart – Shows comparison of student marks
Pie Chart – Shows percentage distribution of marks
Histogram – Shows frequency distribution of marks
Box Plot – Shows spread and variation of marks
Scatter Plot – Shows relationship between age and marks

Business Insights
Average marks of students were calculated.
Highest marks and lowest marks were identified.
Charts were used to visualize student performance.
Relationship between age and marks was analyzed.

Conclusion
This project demonstrates how Exploratory Data Analysis and Business Intelligence techniques can be used to analyze data and generate insights. Python libraries such as pandas and matplotlib help in performing data analysis and visualization efficiently.

How to Run the Project

1. Install required libraries

pip install pandas matplotlib

2. Place the dataset file students.csv in the project folder

3. Run the program

python task2.py
